# 👋 Привет! Меня зовут Вячеслав, я — начинающий QA Engineer

Этот репозиторий — моё учебное портфолио, где я собираю примеры работ по ручному и API-тестированию.  
Рассматриваю позиции стажёра или Junior Manual QA.

## 🔧 Мой стек

*   **Теория:** Понимаю жизненный цикл ПО и тестирования (SDLC, STLC), знаком с принципами ISTQB, применяю базовые техники тест-дизайна (классы эквивалентности, граничные значения).
*   **Инструменты:** Charles Proxy, Postman, Android Studio, DevTools.
*   **Работа с данными и API:** Пишу SQL-запросы (SELECT, JOIN, подзапросы), работаю с JSON и XML. Понимаю клиент-серверную архитектуру и принципы REST API.
*   **Автоматизация (в процессе изучения):** Начал осваивать Python и VS Code для написания автотестов.

## 🛠 Что я изучаю и практикую

*   **API и Бэкенд:** Создаю и поддерживаю коллекции в Postman, настраиваю переменные окружения, использую сниппеты для базовых проверок ответов (статус-коды, структура JSON).
*   **SQL:** Проверяю выборки данных, корректность хранения и связи между таблицами в реляционных БД.
*   **Charles Proxy:** Анализирую HTTP/HTTPS трафик, подменяю заголовки и параметры запросов для локализации проблем на стыке фронтенда и бэкенда.
*   **Тестовая документация:** Составляю чек-листы, тест-кейсы и баг-репорты. Стараюсь, чтобы шаги были воспроизводимыми, а описание бага — полезным для разработчика.
