# QA Portfolio  
Привет! Меня зовут Вячеслав.  

Изучаю тестирование в Яндекс Практикуме.  
Здесь собраны мои учебные проекты, которые отражают мои навыки в ручном тестировании, тестировании API и основы автоматизации.

## 🔧 Навыки и практика

*   **Теория и методологии:** SDLC, STLC, принципы ISTQB, техники тест-дизайна (классы эквивалентности, граничные значения).
*   **API и бэкенд:** Работаю с REST API через Postman — проверяю статус-коды, структуру JSON/XML. Пишу SQL-запросы (SELECT, JOIN, GROUP BY, подзапросы) для проверки данных.
*   **Инструменты:** Charles Proxy (на базовом уровне), Android Studio, DevTools.
*   **Документация:** Составляю чек-листы, тест-кейсы и баг-репорты.
*   **Автоматизация:** Осваиваю Python и Pytest.

## 📈 Мои цели

*   Пройти стажировку или получить позицию Junior QA Engineer.
*   Развиваться в техническом тестировании: API, базы данных, автоматизация.
*   Учиться у опытных коллег и перенимать лучшие практики работы в команде.
