# qa-portfolio
# 👋 Привет! Меня зовут Вячеслав, я — начинающий QA Engineer

Этот репозиторий — моё живое портфолио, демонстрирующее навыки ручного и API-тестирования.  
Ищу позицию стажёра или Junior Manual QA в продуктовую или аутсорсинговую команду.

## 🔧 Мой стек и навыки

*   **Теория:** SDLC, STLC, принципы ISTQB, техники тест-дизайна (КЭ/ГЗ, попарное).
*   **API и Бэкенд:** REST API, JSON, XML. Инструменты: **Postman**, **Charles Proxy**.

> *Моя сильная сторона:* умение анализировать данные, писать SQL-запросы и проверять интеграции через API.

## 🛠 Что я умею на практике

*   **Postman / API:** Создаю коллекции запросов, использую переменные и сниппеты для автотестов на JavaScript (проверка статус-кодов и JSON-схем).
*   **SQL:** Пишу SELECT-запросы, JOINs, подзапросы. Проверяю корректность данных в реляционных БД.
*   **Charles Proxy:** Перехватываю, читаю и модифицирую HTTP/HTTPS трафик для локализации багов на стыке фронтенда и бэкенда.
*   **Тестовая документация:** Составляю понятные и воспроизводимые баг-репорты, чек-листы и тест-кейсы с чёткими шагами и ожидаемыми результатами.

## 📂 Структура портфолио

*   `Test_Documentation/` — Примеры тест-кейсов, чек-листов и баг-репортов.
*   `Postman/` — Экспортированные коллекции и скриншоты тестов.
*   `Charles_Logs/` — Логи и примеры анализа трафика.
*   `SQL/` — SQL-запросы и сценарии проверки данных.
*   `Mobile_Testing/` — Примеры тестирования мобильных приложений.
