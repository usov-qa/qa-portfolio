# Пет-проект: Тестирование мобильного приложения «Калькулятор металлов»

**Цель:** Попрактиковаться в тестировании реального приложения без готовых требований.

**Объект тестирования:** Мобильное приложение для расчёта веса металлопроката (Android, APK). Тестировался один сортамент: **Круг, Сталь, Ст3пс**.

**Подход:** Исследовательское тестирование + техники тест-дизайна (классы эквивалентности и граничные значения).

**Что проверял:**
- Функциональность: логика расчёта массы, длины и количества.
- Негативные сценарии: валидация полей, невалидные символы, граничные значения.
- Интерфейс и вёрстку.
- Юзабилити и UX (жесты, поведение элементов).

**Инструменты:** Android Studio (эмулятор Honor 8, Android 9.0), Google Таблицы, физическая клавиатура ПК.

**Что внутри:**
- [Чек-лист и тест-кейсы](./test-cases.md)
- [Баг-репорты](./bug-reports.md)

**Основные результаты:**
- Найдено 9 багов (от незначительных до критичных).
- Выявлены особенности валидации: на реальном устройстве часть негативных проверок невозможна из-за цифровой клавиатуры.
