# BUG-02: Изменения полей "Имя" и "Занятие" не отображаются на главной странице профиля

**ID:** BUG-02
**Приоритет:** Высокий
**Статус:** Открыт
**Окружение:** Google Chrome 143.0.7499.41, Windows 10 Pro

**Предусловие:**
Открыто окно редактирования профиля.

**Шаги воспроизведения:**
1. Кликнуть на строку ввода "Имя"
2. Ввести имя "Иван"
3. Кликнуть на строку ввода "Занятие"
4. Ввести занятие "Художник"
5. Нажать кнопку "Сохранить"

**Ожидаемый результат:**
На главной странице в профиле отображаются изменения: имя — "Иван" и занятие — "Художник".

**Фактический результат:**
На главной странице в профиле нет изменений.

**Дополнительно:**

*   Этот баг блокирует тестирование кейсов №9, №17, связанных с автоудалением пробелов и валидацией состояния кнопки "Сохранить".
