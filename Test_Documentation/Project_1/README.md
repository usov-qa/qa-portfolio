# Проект 1: «Яндекс Маршруты»
Учебный проект в Яндекс Практикуме
**Задача:** тестирование по готовым тест-кейсам, оформление баг-репортов, ретест исправлений.

## Решение
- Проводил функциональное тестирование веб-приложения по предоставленным тест-кейсам.
- Оформлял баг-репорты в Google Таблицах: описывал шаги, ожидаемый и фактический результат, прикладывал скриншоты.
- Проверял исправленные баги (ретест) и обновлял статусы.
- Составил регрессионный профиль — какие модули затронуты изменениями и что нужно перепроверить.

## Файлы
- [bug_reports.md](./bug_reports.md) — баг-репорты
- [bug_retest.md](./bug_retest.md) — результаты повторного тестирования
- [regression_profile.md](./regression_profile.md) — регрессионный профиль
