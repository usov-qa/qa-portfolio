# Баг-репорты: ретест исправленных дефектов

**Тестировщик:** Вячеслав Усов\
**Тип тестирования:** Повторное тестирование (ретест)

**Описание:**
Данный документ демонстрирует навыки повторного тестирования исправленных дефектов. Проведён ретест багов, обнаруженных на предыдущей итерации тестирования веб-приложения с пользовательским контентом.

---

## Б1: При заходе на главную страницу появляется ошибка

| Параметр | Значение |
| :--- | :--- |
| **Приоритет** | Блокирующий |
| **Статус** | Закрыт |
| **Предусловие** | Приложение не открыто |
| **Шаги воспроизведения** | Выполнить переход в приложение по ссылке тестового стенда |
| **Ожидаемый результат** | Открыт сайт |
| **Фактический результат** | Отображается ошибка 502 Bad Gateway. Переход на сайт не выполнен |
| **Комментарий** | Ошибка исправлена |
