# Этап 1. Тест-анализ

## Декомпозиция логики полей формы

### Поле ввода часов
- Отрицательные числа
- Числа от 0 до 23
- Числа от 24 или больше
- Нули перед однозначным числом
- Поле заполнено
- Поле не заполнено
- Валидация: сообщение об ошибке, рамка поля подсвечивается красным

### Поле ввода минут
- Отрицательные числа
- Числа от 0 до 59
- Числа от 60 или больше
- Нули перед однозначным числом
- Поле заполнено
- Поле не заполнено
- Валидация: сообщение об ошибке, рамка поля подсвечивается красным

### Поле ввода адреса (для полей «Откуда» и «Куда»)
- Адреса не из списка доступных
- Фиксированные адреса
- Пробел до адреса
- Пробел после адреса
- Поле заполнено
- Поле не заполнено
- Валидация: сообщение об ошибке, рамка поля подсвечивается красным

### Расчет времени и стоимости поездки на собственном автомобиле
- Длина маршрута
- Время поездки по маршруту
- Стоимость поездки по маршруту
