# Чек-лист регрессионного тестирования: форма редактирования профиля

**Тестировщик:** Вячеслав Усов
**Дата:** 2026-04-23
**Тип тестирования:** Регрессионное

**Описание тестируемой функциональности:**
Форма редактирования профиля пользователя с полями "Имя" и "Занятие".
- **Имя:** текстовое поле с ограничением по длине (вероятный диапазон 2-40 символов, проверен в рамках чек-листа).
- **Занятие:** текстовое поле с ограничением по длине (вероятный диапазон 2-200 символов, проверен в рамках чек-листа).
- **Кнопка "Сохранить":** активируется только при корректном заполнении полей.

**Примечание:** Данный артефакт демонстрирует владение техниками тест-дизайна (граничные значения, негативные проверки) и навыки документирования.
