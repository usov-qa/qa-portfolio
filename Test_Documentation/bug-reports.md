## BUG-02: Изменения полей "Имя" и "Занятие" не отображаются на главной странице профиля

| Параметр | Значение |
| :--- | :--- |
| **Приоритет** | Высокий |
| **Статус** | Открыт |
| **Предусловие** | Открыто окно редактирования профиля |
| **Шаги воспроизведения** | 1. Кликнуть на строку ввода "Имя"<br>2. Ввести "Иван"<br>3. Кликнуть на строку ввода "Занятие"<br>4. Ввести "Художник"<br>5. Нажать кнопку "Сохранить" |
| **Ожидаемый результат** | На главной странице в профиле отображаются изменения: имя — "Иван" и занятие — "Художник" |
| **Фактический результат** | На главной странице в профиле нет изменений |

## BUG-07: В поле "Занятие" невозможно ввести количество символов больше допустимого

| Параметр | Значение |
| :--- | :--- |
| **Приоритет** | Низкий |
| **Статус** | Открыт |
| **Предусловие** | Открыто окно редактирования профиля |
| **Шаги воспроизведения** | 1. Кликнуть на строку ввода "Занятие"<br>2. Ввести значение, заведомо превышающее 200 символов |
| **Ожидаемый результат** | Под полем отображается красный текст об ошибке "Недопустимое количество символов" |
| **Фактический результат** | Под полем не отображается красный текст об ошибке. Кроме того, поле вообще не даёт ввести более 200 символов, и никакой обратной связи от системы нет |
