# Проект 2: Тестирование сервиса «Яндекс Маршруты»

Учебный проект в Яндекс Практикуме.  
**Задача:** полный цикл ручного тестирования (от анализа требований до баг-репортов).

## Что я делал
- Провёл тест-анализ: декомпозировал логику полей формы.
- Применил техники тест-дизайна: классы эквивалентности и граничные значения.
- Составил тест-кейсы и провёл функциональное тестирование.
- Оформил баг-репорты по найденным дефектам.

## Файлы
- [test-analysis.md](./test-analysis.md) — тест-анализ и декомпозиция
- [equivalence-classes.md](./equivalence-classes.md) — классы эквивалентности и граничные значения
- [test-cases.md](./test-cases.md) — тест-кейсы
- [bug-reports.md](./bug-reports.md) — баг-репорты

## Инструменты
Google Таблицы, Яндекс Маршруты (тестовый стенд).
